# Third Person Controller + Animations
### Basic Third Person Character Controller for Unity 6

---

# 🇺🇸 English

## Overview

This asset provides a lightweight third-person controller for Unity 6.

### Included Features

- Third Person Movement
- Camera Relative Movement
- Smooth Character Rotation
- Double Tap Sprint
- Crouch System
- Double Jump
- Coyote Time
- Gravity System
- Fall Detection
- Camera FOV Effects
- Wind Shake Camera Effect
- Foot IK for Idle Animations
- Animator Integration
- Cinemachine Support
- Unity Input System Support

---

## Requirements

- Unity 6
- Input System Package
- Cinemachine Package

---

## Installation

1. Import the package.
2. Open the Demo Scene.
3. Drag the Player Prefab into your scene.
4. Make sure Input System is installed.
5. Make sure Cinemachine is installed.
6. Delete the default Main Camera from your scene.

---

## Animator Parameters

### Bools

- Walking
- Running
- IsGrounded
- IsCrouching
- CrouchBlock

### Triggers

- Jump
- DoubleJump
- Fall

---

## Foot IK Setup

Enable **IK Pass** on the Animator Layer.

---

## Camera Wind Shake Setup

Add the following component to your Cinemachine Camera:

- CinemachineBasicMultiChannelPerlin

---

## Support

For support, please contact through the Unity Asset Store.
